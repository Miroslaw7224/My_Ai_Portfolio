Code Sensei

Interaktywna aplikacja webowa do analizy i refaktoryzacji kodu z użyciem AI. Integracja OpenAI GPT-4o, generowanie audio (TTS), historia analiz i zarządzanie kosztami tokenów.

// LIVE DEMO ↗ // GITHUB REPO ↗

Problem → Rozwiązanie

Problem: Programiści często potrzebują szybkiego wyjaśnienia działania kodu, refaktoryzacji lub odpowiedzi na pytania dotyczące fragmentów kodu. Ręczna analiza jest czasochłonna, a dokumentacja często nieaktualna.

Rozwiązanie: Code Sensei to aplikacja webowa, która wykorzystuje OpenAI GPT-4o do automatycznej analizy kodu w czasie rzeczywistym. Użytkownik wkleja kod, wybiera typ analizy (opis ogólny, szczegółowy, Q&A, refaktoryzacja) i otrzymuje natychmiastową odpowiedź. Dodatkowo może wygenerować wyjaśnienie audio (TTS) i przeglądać historię wszystkich analiz z podsumowaniem kosztów.

Stack Technologiczny

Python 3.x Streamlit OpenAI API (GPT-4o) Text-to-Speech (TTS) SQLite Pandas Streamlit Cloud Git / GitHub

Kluczowe Funkcje

Analiza kodu AI
Generowanie opisów ogólnych i szczegółowych (blokowych lub linia po linii) w języku polskim lub angielskim.
Wyjaśnienia audio (TTS)
Automatyczne generowanie nagrań audio z opisów kodu z wyborem głosu (Alloy, Nova, Shimmer).
Tryb Q&A
Zadawanie pytań o kod i otrzymywanie kontekstowych odpowiedzi od AI.
Refaktoryzacja
Automatyczne przekształcenie kodu na bardziej czytelny z komentarzem wyjaśniającym zmiany.
Historia analiz
Przegląd wszystkich wygenerowanych opisów z filtrowaniem i eksportem do CSV.
Zarządzanie kosztami
Automatyczne przeliczanie tokenów OpenAI na koszt w PLN z podsumowaniem statystyk.

Kluczowe Wyzwania Techniczne

  • Integracja OpenAI API: Implementacja wielotrybowego promptowania (opis ogólny, szczegółowy, Q&A, refaktoryzacja) z obsługą różnych modeli GPT.
  • Generowanie audio TTS: Konwersja opisów tekstowych na pliki MP3 z możliwością odtwarzania i pobierania.
  • Zarządzanie historią: Projektowanie schematu bazy SQLite do przechowywania analiz, nagrań audio i statystyk kosztów.
  • Obliczanie kosztów: Automatyczne przeliczanie tokenów (prompt + completion) na koszt w PLN z uwzględnieniem różnych modeli i cen.
  • UX w Streamlit: Tworzenie intuicyjnego interfejsu z sidebar, zakładkami, przyciskami audio i eksportem CSV.
  • Deployment na Streamlit Cloud: Konfiguracja secrets (API keys), zarządzanie zależnościami i optymalizacja wydajności.

Jak to działa?

1. Użytkownik wkleja kod → Wprowadza klucz OpenAI API i wkleja fragment kodu do analizy.

2. Wybór trybu analizy → Wybiera język (polski/angielski) i typ opisu (ogólny/szczegółowy).

3. Generowanie opisu → AI analizuje kod i generuje wyjaśnienie działania.

4. Opcjonalne audio → Użytkownik może wygenerować nagranie audio z opisem.

5. Historia i statystyki → Wszystkie analizy są zapisywane w bazie SQLite z możliwością przeglądania, filtrowania i eksportu.

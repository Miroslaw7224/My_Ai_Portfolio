Titanic Survival Prediction

Analiza przeżywalności pasażerów Titanica z wykorzystaniem różnych algorytmów Machine Learning. Eksploracyjna analiza danych (EDA), wizualizacje statystyczne i modelowanie predykcyjne.

// PEŁNA ANALIZA ↗ // GITHUB REPO ↗

Problem → Rozwiązanie

Problem: Katastrofa Titanica w 1912 roku pozostawiła wiele pytań o czynniki wpływające na przeżywalność pasażerów. Czy wiek, płeć, klasa podróży miały znaczenie? Jak można przewidzieć szanse przeżycia na podstawie dostępnych danych?

Rozwiązanie: Projekt wykorzystuje techniki Data Science do analizy danych 891 pasażerów. Poprzez eksploracyjną analizę danych (EDA), wizualizacje i modelowanie Machine Learning, odkrywam kluczowe wzorce i buduję model predykcyjny klasyfikacji binarnej (przeżył/nie przeżył).

Stack Technologiczny

Python 3.x Pandas NumPy Scikit-learn Matplotlib Seaborn Plotly Jupyter Notebook

Kluczowe Elementy Analizy

Czyszczenie danych
Analiza braków w danych (Age, Cabin), obsługa wartości null i przygotowanie datasetu do modelowania.
Eksploracja zmiennych
Rozkłady wieku, płci, klasy podróży. Histogramy, wykresy słupkowe i pudełkowe.
Analiza przeżywalności
Badanie czynników wpływających na szanse przeżycia: płeć, wiek, klasa, rodzina.
Macierze korelacji
Heatmapy pokazujące zależności między cechami numerycznymi.
Feature Engineering
Tworzenie nowych zmiennych (FamilySize, IsAlone) dla lepszej predykcji.
Modelowanie ML
Porównanie algorytmów: Logistic Regression, Random Forest, SVM, Decision Trees.

Kluczowe Wnioski

  • Płeć: Kobiety miały znacznie wyższe szanse przeżycia (~74%) niż mężczyźni (~19%) - polityka "kobiety i dzieci najpierw".
  • Klasa podróży: Pasażerowie 1. klasy mieli wyższy wskaźnik przeżycia (~63%) niż 3. klasy (~24%).
  • Wiek: Dzieci (poniżej 10 lat) miały wyższe szanse przeżycia niż dorośli.
  • Rodzina: Pasażerowie podróżujący z rodziną (2-4 osoby) mieli lepsze szanse niż osoby samotne lub w bardzo dużych grupach.
  • Braki w danych: Kolumna Age miała ~20% braków, Cabin ~77% - wymagało to odpowiedniej strategii imputacji.
  • Model ML: Random Forest osiągnął najlepszą dokładność (~82%) w predykcji przeżywalności.

Zastosowania

Ten projekt pokazuje kompletny pipeline Data Science: od czyszczenia danych, przez EDA i wizualizacje, po budowę modelu predykcyjnego. Techniki użyte w tym projekcie można zastosować do:

Klasyfikacja binarna
Przewidywanie wyników tak/nie w różnych domenach (churn, fraud detection).
Analiza historyczna
Badanie danych historycznych i wyciąganie wniosków społecznych.
Feature Engineering
Tworzenie nowych zmiennych z istniejących danych dla lepszych predykcji.

Titanic Survival Prediction

Analysis of Titanic passenger survival using multiple machine learning algorithms. Exploratory data analysis (EDA), statistical visualizations, and predictive modeling.

// FULL ANALYSIS ↗ // GITHUB REPOSITORY ↗

Problem -> Solution

Problem: The Titanic disaster in 1912 raised many questions about factors affecting passenger survival. Did age, sex, and travel class matter? How can survival chances be predicted from available data?

Solution: The project applies Data Science techniques to analyze data from 891 passengers. Through exploratory data analysis (EDA), visualizations, and machine learning modeling, I uncover key patterns and build a binary classification model (survived/did not survive).

Tech Stack

Python 3.x Pandas NumPy Scikit-learn Matplotlib Seaborn Plotly Jupyter Notebook

Key Analysis Components

Data cleaning
Missing-data analysis (Age, Cabin), null handling, and dataset preparation for modeling.
Variable exploration
Distributions of age, sex, and travel class. Histograms, bar charts, and box plots.
Survival analysis
Investigation of factors influencing survival: sex, age, class, family.
Correlation matrices
Heatmaps showing relationships between numerical features.
Feature Engineering
Creating new features (FamilySize, IsAlone) for better prediction.
ML modeling
Algorithm comparison: Logistic Regression, Random Forest, SVM, Decision Trees.

Key Findings

  • Sex: Women had much higher survival chances (~74%) than men (~19%) - due to the "women and children first" policy.
  • Travel class: First-class passengers had a higher survival rate (~63%) than third-class passengers (~24%).
  • Age: Children (under 10) had higher survival chances than adults.
  • Family: Passengers traveling with family (2-4 people) had better outcomes than solo travelers or very large groups.
  • Missing data: The Age column had ~20% missing values, Cabin ~77% - requiring a proper imputation strategy.
  • ML model: Random Forest achieved the best accuracy (~82%) for survival prediction.

Applications

This project shows a complete Data Science pipeline: from data cleaning, through EDA and visualizations, to building a predictive model. The techniques used in this project can be applied to:

Binary classification
Predicting binary outcomes across domains (churn, fraud detection).
Historical analysis
Analyzing historical data and extracting social insights.
Feature Engineering
Creating new features from existing data for better predictions.
